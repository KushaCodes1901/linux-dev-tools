# Linux Dev Tools

A collection of useful Linux shell scripts for development setup, maintenance, backups, cache cleanup, and system information.

## Included Scripts

- `install-dev-tools.sh` – installs common developer packages
- `update-system.sh` – updates system packages
- `backup-home.sh` – creates a backup of common user folders
- `clean-cache.sh` – clears user cache and temporary files
- `system-info.sh` – displays useful system information

## Skills Demonstrated

- Linux terminal usage
- Bash scripting
- System maintenance
- Automation
- Cross-distro package management basics
